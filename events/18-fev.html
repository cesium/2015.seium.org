Palestra

API Driven Development

11h30 - DI A2

Descrição

Com o surgimento de várias plataformas e web services, é necessário complementar qualquer serviço que queiramos oferecer com uma API digna. Nesta palestra de uma hora e meia, os nossos parceiros da Seegno irão mostrar-te os conceitos básicos de uma API e como devem ser construídas.

Descrição

À semelhança do ano passado, haverá uma reunião de duas horas para debater o estado atual do curso. Gostaríamos de contar com presença de todos os alunos, para podermos ouvir a opinião de cada um e assim ajudar a melhorar a licenciatura. A reunião contará com a presença da direção de curso.

Descrição

Depois do sucesso da edição passada, a Corrida Social CeSIUM está de volta!
